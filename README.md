# Mobile-Sales-POWER-BI-DASHBOARD

## Overview
This project is an interactive Power BI dashboard designed to analyze mobile phone sales data and uncover meaningful business insights. It focuses on sales performance, customer behavior, and transaction trends using clear and intuitive visualizations.

## Objective
To provide a consolidated view of mobile sales data that helps stakeholders:
- Track overall sales performance
- Identify monthly trends
- Compare brand and model performance
- Understand customer preferences and payment behavior

## Tools & Technologies
- Power BI (Data Modeling & Visualization)
- Microsoft Excel

## Dataset Description
The dataset includes mobile sales transaction data with the following attributes:
- Date (Month-wise analysis)
- Brand and Mobile Model
- City / Location
- Quantity Sold
- Sales Amount
- Payment Method
- Customer Ratings

## Key Insights Delivered
- Total Sales, Quantity Sold, and Number of Transactions
- Month-wise sales trends using slicers
- City-wise sales distribution
- Brand and model-wise sales comparison
- Payment method usage analysis
- Customer rating overview

## Dashboard Features
- Interactive month slicer for dynamic analysis
- Clean and professional dashboard layout
- Multiple visuals for comparative insights
- User-friendly design for quick decision-making

## Project Type
Academic / Internship / Portfolio Project
